# 📦 SwiftRoute – IBM Cognos Logistics Dashboard

## 📖 Project Overview
SwiftRoute is a web-based logistics dashboard designed to monitor, analyze, and optimize shipment operations in real time. It integrates IBM Cognos Analytics with a custom frontend to provide powerful business insights.

## 🎯 Objectives
- Monitor shipment performance  
- Analyze delivery status (On-time, Delayed, Failed)  
- Track fuel efficiency and costs  
- Provide real-time analytics using IBM Cognos  

## 🚀 Features
- Login system (demo authentication)  
- KPI dashboard (shipments, ratings, fuel cost)  
- Interactive charts using Chart.js  
- Dynamic shipment table with search  
- Embedded IBM Cognos dashboard  
- Responsive design  

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript  
- Chart.js  
- IBM Cognos Analytics  
- Font Awesome  

## 📊 IBM Cognos Integration
IBM Cognos dashboard is embedded using an iframe to display real-time logistics analytics and reports.
